# Online Shopping Database System (CSE370)

## Overview
This project was developed as part of CSE370 – Database Systems.
It implements a complete online shopping management system using relational database design principles and SQL queries, supported by a PHP-based web interface.

The system manages customers, employees, products, suppliers, orders, and inventory while supporting search and analytical queries.

## Features
- Customer management
- Supplier management
- Product and inventory management
- Order and suborder handling
- Search functionality
- Sales and revenue analysis using SQL queries

## Database Design
- Enhanced Entity-Relationship (EER) diagram
- Relational schema design
- Normalized tables with primary and foreign keys

## Technologies Used
- Backend: PHP
- Database: MySQL / SQL
- Frontend: HTML, CSS, JavaScript

## Project Structure
css/     -> Frontend styling  
js/      -> Client-side scripts  
img/     -> UI images  
php/     -> Backend PHP files (CRUD operations)  
sql/     -> Database schema and queries  
report/  -> Project report  

## How to Run
1. Import SQL files from the sql folder into MySQL
2. Place the project folder inside htdocs (XAMPP)
3. Start Apache and MySQL
4. Open the main PHP file in a browser
